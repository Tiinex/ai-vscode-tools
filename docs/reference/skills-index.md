# Skills Index

This page is a navigational index for the repo's skill layer.

This is mainly a browsing page, not a page you need to study end to end.

In the current checkout, the skill family is historical and parked rather than live in the workspace root.
That means this page is mainly an index into the parked support lineage, not a promise that those skills are active runtime input right now.

For what skills are and how they relate to other artifact types, read [skill-system.md](skill-system.md).
For contributor-facing authoring guidance, read [../guides/creating-skills.md](../guides/creating-skills.md).
